Custom Number Format Codes for Display Control

Custom number formats control how values appear in cells without altering underlying data, enabling formulas and charts to function normally. The technique uses semicolon-separated sections (Positive;Negative;Zero;Text) with codes to scale numbers, apply conditional colors, replace values with text, hide zeros, add symbols, and format patterns like phone numbers.
