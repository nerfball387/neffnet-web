GROUPBY function for dynamic formula-based data summarization

GROUPBY creates formula-driven summaries that function like PivotTables but return spilled arrays that update dynamically. It accepts group-by columns, values, aggregation functions (including custom LAMBDA), and optional parameters for headers, subtotals, sorting, and filtering—eliminating the need for static pivot objects while enabling direct formula chaining and conditional formatting.

First Normal Form for Excel data structure

First Normal Form (1NF) is a database normalization principle applied to Excel data organization that ensures each column contains atomic (indivisible) values and eliminates repeating groups, creating a flat table structure. Implementing 1NF simplifies downstream analysis, reduces formula complexity, and enables proper use of Excel's data tools like pivot tables and filters without requiring workarounds.
