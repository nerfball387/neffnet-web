Sorting Categories Correctly in GROUPBY Function

GROUPBY sorts text alphabetically by default, which breaks categorical data with natural ordering (months, sizes, etc.). This technique demonstrates a method to enforce correct category order in GROUPBY results so reports display logically rather than alphabetically.
