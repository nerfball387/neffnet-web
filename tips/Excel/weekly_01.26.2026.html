GROUPBY Function for Dynamic Data Aggregation

The GROUPBY function creates dynamic, formula-based data summaries that function like PivotTables but update automatically when source data changes. It accepts grouping columns, value ranges, aggregation functions (including LAMBDA), and optional parameters for headers, subtotals, sorting, and filtering—all returning spilled arrays that can feed into other formulas.

First Normal Form for Clean Data Structure

First Normal Form (1NF) is a data organization principle that eliminates repeating groups and ensures each cell contains atomic (indivisible) values. Implementing 1NF reduces formula complexity and improves data analysis by establishing a clean, flat table structure where each row represents a single entity.
