Transform Sample Sheet Pattern for Power Query Multi-Sheet Transformations

Power Query's Transform Sample File feature—which automatically applies changes to all files in a folder—can be replicated for combining multiple worksheets within a single workbook. The technique uses parameters and custom functions to apply consistent transformations across multiple sheets without requiring advanced M code.
