TRIMRANGE Function and Trim References Operator

TRIMRANGE is a new Excel 365 function that removes empty rows and columns from a range, with the Trim References operator providing an alternative syntax. The function addresses use cases where data ranges contain blank cells that complicate formulas, particularly useful in dynamic array contexts where clean, contiguous ranges improve downstream formula performance.
